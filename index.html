Installation Guide

Method 1: WordPress Admin

  1. Download the flux-seo-scribe-craft-wordpress-plugin.zip file
  2. Go to WordPress Admin → Plugins → Add New
  3. Click "Upload Plugin" and select the zip file
  4. Click "Install Now" and then "Activate"
  5. Navigate to "Flux SEO" in your WordPress admin menu

Method 2: Manual Installation

  1. Extract the zip file to your computer
  2. Upload the flux-seo-scribe-craft folder to your /wp-content/plugins/ directory
  3. Go to WordPress Admin → Plugins
  4. Find "Flux SEO Scribe Craft" and click "Activate"
  5. Navigate to "Flux SEO" in your WordPress admin menu
