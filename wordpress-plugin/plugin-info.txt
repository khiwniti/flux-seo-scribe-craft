=== INSTALLATION INSTRUCTIONS ===

1. Upload the entire 'flux-seo-scribe-craft' folder to your WordPress plugins directory:
   /wp-content/plugins/flux-seo-scribe-craft/

2. Activate the plugin through the WordPress admin:
   - Go to Plugins > Installed Plugins
   - Find "Flux SEO Scribe Craft"
   - Click "Activate"

3. Access the plugin:
   - Admin Dashboard: Go to "SEO Scribe Craft" in the WordPress admin menu
   - Shortcode: Use [flux_seo_scribe_craft] on any page or post
