== Description ==

Flux SEO Scribe Craft is a comprehensive WordPress plugin that provides professional SEO optimization tools, content generation capabilities, and advanced analytics.

= Key Features =

* **Content Analyzer** - Analyze your content for SEO optimization opportunities with real-time scoring and suggestions.
* **Blog Generator** - Generate SEO-optimized blog posts with AI assistance.
* **Advanced Analytics** - Track and monitor your SEO performance with comprehensive reporting.
* **WordPress Integration** - Seamlessly integrates with your WordPress admin dashboard.
* **Shortcode Support** - Embed SEO tools anywhere with simple shortcodes.

= Who Is This For? =

* Content creators looking to optimize their articles for search engines
* Website owners wanting to improve their SEO performance
* Marketers seeking data-driven insights for content strategy
* Bloggers who want to generate SEO-friendly content quickly

== Installation ==

1. Upload the `flux-seo-scribe-craft` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings at 'Flux SEO > Settings'
4. Access the main dashboard at 'Flux SEO > Dashboard'
5. Use the shortcode `[flux_seo_scribe_craft]` to embed tools on any page or post

== Frequently Asked Questions ==

= Does this plugin require an API key? =

Yes, for AI-powered features like content generation, you'll need to enter an API key in the settings page.

= Can I use this plugin with any WordPress theme? =

Yes, Flux SEO Scribe Craft is designed to work with any WordPress theme.

= Is the plugin mobile-friendly? =

Yes, all interfaces are fully responsive and work well on mobile devices.

= How do I use the shortcode? =

Simply add `[flux_seo_scribe_craft]` to any page or post. You can customize the dimensions with parameters like `[flux_seo_scribe_craft height="600px" width="90%"]`.
